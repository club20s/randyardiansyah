Algorithmic Reduction

This project investigates classic image dithering algorithms as computational aesthetics. Through the implementation of error diffusion, ordered thresholding, and stochastic noise techniques, the system demonstrates how early digital imaging constraints can be recontextualized as contemporary visual expression.
